    • Method Overriding

    When a method in a subclass has the same name, same parameters or signature, and same return type(or sub-type) as a method in its super-class, then the method in the subclass is said to override the method

    In other words, If a subclass provides the specific implementation of the method that has been declared by one of its superclass, it is known as method overriding and the technique is called polymorphism

    The word polymorphism means having many forms.

    The word polymorphism means having many forms.

    • Method overriding is used to provide the specific implementation of a method which is already provided by its superclass.
    • Method overriding is used for runtime polymorphism

    Rules for Method Overriding

    1. The method must have the same name as in the parent class
    2. The method must have the same parameter as in the parent class.
    3. They must inherit some class.
    4. We cannot override the method declared as final and static.
    Example:-
        class Animal {
      public void displayInfo() {
        System.out.println("I am an animal.");
      }
    }

    class Dog extends Animal {
      public void displayInfo() {
        System.out.println("I am a dog.");
      }
    }

    class Main {
      public static void main(String[] args) {
        Dog d1 = new Dog();
        d1.displayInfo();
      }
    }
    Output:
        I am a dog.

    To access the method of the superclass from the subclass, we use the super keyword.

    Example:-
        class Animal {
      public void displayInfo() {
        System.out.println("I am an animal.");
      }
    }

    class Dog extends Animal {
      public void displayInfo() {
        super.displayInfo();
        System.out.println("I am a dog.");
      }
    }

    class Main {
      public static void main(String[] args) {
        Dog d1 = new Dog();
        d1.displayInfo();
      }
    }
    Output:
    I am an animal.
I am a dog.

    Note:- constructors in Java are not inherited. Hence, there is no such thing as constructor overriding in Java.

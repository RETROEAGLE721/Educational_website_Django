    • Method

    A method is a block of code that performs a specific task.

    A method that divides a complex problem into smaller chunks makes your program easy to understand and reusable.

    A method is a block of code which only runs when it is called.

    Return Type: Return type is a data type that the method returns. It may have a primitive data type, object, collection, void, etc. If the method does not return anything, we use the void keyword.

    Method Name: It is a name that is used to define the name of a method.

    Parameter List: It is the list of parameters separated by a comma and enclosed in the pair of parentheses. It contains the data type and variable name. If the method has no parameter, leave the parentheses blank.

    Method Body: It is a part of the method declaration. It contains all the actions to be performed. It is enclosed within the pair of curly braces.

                             public class Main {
                           public void myMethod(int a) {
                             // code to be executed
                            }
                          }

    There are two types of methods in Java:

    • Predefined Method
    • User-defined Method

    1.Predefined Method:

    The method that is already defined in the Java class libraries is known as predefined methods. It is also known as the standard library method or built-in method. We can directly use these methods just by calling them in the program at any point. Some predefined methods are length(), equals(), compareTo(), sqrt(), etc.


    Example:-
      public class LengthExample
    {  
      public static void main(String args[])
    {

    String s1="ABCD";  
    System.out.println("string length is: "+s1.length()); 
    }
  }
    Output:
    4

    2.User-defined Method:


    The method written by the user or programmer is known as a user-defined method. These methods are modified according to the requirement.

    Example:
        class add{
      public int addNumbers(int a, int b) {
      int sum = a + b;
      return sum;
    }

    Class Main{
      public static void main(String[] args) { 
        int num1 = 25,num2 = 15;
        add obj = new add();
        int result = obj.addNumbers(num1, num2);
        System.out.println("Sum is: " + result);
      }
    }
    Output:
        Sum is: 40

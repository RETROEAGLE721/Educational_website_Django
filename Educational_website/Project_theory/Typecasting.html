    • Type Casting :

    We often encounter situations where there is a need to store a value of one type into a variable of another type.

    In such situations,we must cast the value to be strored by proceeding it with the type name in parentheses.

    Type variable1 = (type) variable2

    The process of converting one data type to another is called casting.

    int m = 50;
    byte n = (byte)m;
    long count = (long)m;

    No. Form To
    1 byte short, char, int, long, float, double
    2 short int, long, float, double
    3 char int, long, float, double
    4 int long, float, double

    • Scanner

    The class Scanner enables reading data from the keyboard, a String object, or a text file using the characters The source code for Scanner is available as part of JDK but not part of java.lang. Therefore, to write a source code for a program that uses Scanner, the source code of Scanner must be included. To include Scanner, an import statement must be used.

      1.   import java.util.*;
  2.   class FooBar 
  3.   {
  4.   ... 
  5.   }

    Scanner, accept more than one kind of parameter specification.Three

    Scanner :

    1. String object
    2. System.in
    3. file object

    Below are the way to initialise Scanner

    1. Scanner strScanner , fileScanner , keyboard;
    2. strScanner = new Scanner( "My GPA is 4.00!" );
    3. fileScanner = new Scanner( new File( "theFile.txt" ) );
    4. keyboard = new Scanner( System.in );

    The first Scanner object strScanner scans the String literal "My GPA is 4.00!",the second Scanner object keyboard scans input from the keyboard and the last Scanner object fileScanner scans the whole file "theFile.txt".

    The asterisk, meaning “everything”, is called the wildcard.

    Scanner offers a variety of token reading methods: next, nextBoolean, nextByte, nextDouble, nextFloat, nextInt, nextLine, nextLong, and nextShort.


    Name Return Type Action
    next String Obtains the next token
    nextBoolean boolean Obtains the next boolean token.
    nextByte byte Obtains the next byte token.
    nextDouble double Obtains the next double token.
    nextFloat float Obtains the next float token.
    nextInt int Obtains the next int token.
    nextLong long Obtains the next long token.
    nextShort short Obtains the next short token.
    nextLine string Obtains the String before the next newline.
    hasNext boolean Checks whether or not the next token exists.
    Example:-
        import java.util.Scanner;
    public class BeyondLimit 
    { 
      public static void main( String[] args ) 
        { 
          String aToken , tokens = new String( "My age is 20" ); 
          Scanner keyboard = new Scanner( tokens ); 
          aToken = keyboard.next(); 
          System.out.println( aToken ); 

          aToken = keyboard.next(); 
          System.out.println( aToken ); 

          aToken = keyboard.next(); 
          System.out.println( aToken ); 

          aToken = keyboard.next(); 
          System.out.println( aToken ); 

          aToken = keyboard.next(); 
          System.out.println( aToken ); 
      } 
    }

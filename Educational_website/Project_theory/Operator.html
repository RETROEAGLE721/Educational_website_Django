    • Operators

    Operators in java programming language are special symbols that perform specific operations on one,two or three operands, and then return a result.

    1. Increment/Decrement Operators (++/--) :

    Variables can be incremented or decremented by 1 using the ++and --operators,respectively.

    Post Increment/Decrement Operators

    post Increment = a++
post-decrement = a--
    Example :
    int a=10;
a++; // a now quals 11
a--; // a now quals 10 again

    2. Pre Increment/Decrement Operators :

    pre-Increment = ++a
pre-decrement = --a
    Example :
    int a=10;
++a; // a now quals 10
--a; // a now quals 9

    3. The Conditional Operator(? :)

    It is the shorter form of if else condition.

    Here first we have to write conditional statement. If the condition is true than it will execute the statement befor colon and if the statement is false than it will execute statement after colon

    Syntax :
    Syntax :
{condition-to-evaluate}?{statement-executed-on-true}:{statement-executed-on-false}

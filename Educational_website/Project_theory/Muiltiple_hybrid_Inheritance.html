    Java does not support Muiltiple Inheritance

    To reduce the complexity and simplify the language, multiple inheritance is not supported in java.In Java, we can achieve multiple inheritance only through Interfaces.

    Example:-
        class A
    {  
      void msg(){System.out.println("Hello");}  
    } 

    class B
    {  
      void msg(){System.out.println("Welcome");
    }  

    class C extends A,B
    {
      public static void main(String args[])
      {  
        C obj=new C();  
        obj.msg();//Now which msg() method would be invoked?  
      }  
    }
    Output:
        Compile Time Error


    Hybrid Inheritance

    the meaning of hybrid (mixture) is made of more than one thing. In Java, the hybrid inheritance is the composition of two or more types of inheritance. The main purpose of using hybrid inheritance is to modularize the code into well-defined classes. It also provides the code reusability.

    Example:-
        class A
    {  
      void msg(){System.out.println("Hello");}  
    } 

    class B extends A
    {  
    }  

    class C extends A
    {
      public static void main(String args[])
      {  
        C obj=new C();  
        B obj1=new B();  
        obj.msg();//Now which msg() method would be invoked?  
        obj1.msg();//Now which msg() method would be invoked?  
      }  
    }
    Output:
      Hello
  Hello
